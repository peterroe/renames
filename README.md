## renames

## Define 

* `()` : captures the expression.
* `*` : match zero or more characters.
* `+` : match one or more characters.
* `?` : match zero or one character.
* `[a-z]`: match a character in the range.

## example 

```js
'hello_test.js' // => 'hello.test.js'
// '([a-z]*)(_)test.js' '([a-z]*)(.)test.js'


```
